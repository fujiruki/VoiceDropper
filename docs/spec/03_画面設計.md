# 03. 画面設計

## メインウィンドウレイアウト

```
┌──────────────────────────────────────────┐
│                                          │
│  ┌──────────────────────────────────┐   │
│  │                                  │   │
│  │   テキスト入力欄（複数行）        │   │
│  │                                  │   │
│  └──────────────────────────────────┘   │
│                                          │
│  [✕][⚙][▼履歴][☑最前面][📋][🗑][🎤]    │
└──────────────────────────────────────────┘
```

- タイトルバーなし（-Caption）
- 空き領域をドラッグでウィンドウ移動

## ウィンドウ仕様

| 項目 | 値 |
|:--|:--|
| 初期サイズ | 幅 460px × 高さ 200px |
| リサイズ | 可能 |
| 最前面表示 | 初期ON（チェックボックスで切替） |
| タイトルバー | 非表示 |
| ドラッグ移動 | 空き領域どこでも可能 |

## ボタンバー（左から順）

| 要素 | 幅 | 説明 |
|:--|:--|:--|
| ✕ 閉じる | 30px | アプリ終了 |
| ⚙ 設定 | 40px | 設定画面を開く（Segoe MDL2 Assets U+E713） |
| ▼履歴 | 70px | コピー履歴ドロップダウン |
| ☑ 最前面 | 62px | 常時最前面のON/OFF |
| 📋 コピー | 40px | 手動コピー（Segoe MDL2 Assets U+E8C8） |
| 🗑 クリア | 40px | 履歴保存+クリア |
| 🎤 マイク | 40px | Win+H音声入力起動 |

- ボタン間余白: 2px
- 右余白: 2px
- 下余白: 2px

## コピー通知の表現

- コピー成功時：テキスト入力欄の背景色を `#C8F7C5`（薄緑）にフラッシュ → 300ms後に白に戻す
