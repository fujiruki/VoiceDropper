# 03. 画面設計

## ウィンドウレイアウト

```
┌─ VoiceDropper ──────── [×] ─┐
│                              │
│  ┌──────────────────────┐   │
│  │                      │   │
│  │   テキスト入力欄      │   │
│  │   （複数行）          │   │
│  │                      │   │
│  └──────────────────────┘   │
│                              │
│  [▼ 履歴] [📋 コピー] [🗑 クリア] │
│                              │
└──────────────────────────────┘
```

## ウィンドウ仕様

| 項目 | 値 |
|:--|:--|
| 初期サイズ | 幅 300px × 高さ 200px |
| 最小サイズ | 幅 200px × 高さ 150px |
| 初期位置 | 画面右下 |
| リサイズ | 可能 |
| 最前面表示 | 常時ON |
| 透過度 | 不透明（必要なら将来対応） |

## UI要素

| 要素 | 種類 | 説明 |
|:--|:--|:--|
| テキスト入力欄 | Edit（複数行） | メインの入力エリア。音声入力(Win+H)の受け皿 |
| コピーボタン | Button | 手動でクリップボードにコピーする（自動コピーとは別に用意） |
| 履歴ボタン | Button（▼） | コピーボタンの左に配置。押すとドロップダウンメニューで直近10件の履歴を表示 |
| クリアボタン | Button（🗑） | コピーボタンの右に配置。履歴に保存してからメモ欄をクリア（クリップボードにはコピーしない） |

## コピー通知の表現

- コピー成功時：テキスト入力欄の背景色を一瞬 `#C8F7C5`（薄緑）にフラッシュ → 300ms後に白に戻す
