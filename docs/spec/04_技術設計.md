# 04. 技術設計

## 技術スタック

| 項目 | 選定 | 理由 |
|:--|:--|:--|
| 言語 | AutoHotkey v2 | Windows常駐ツールに最適。ウィンドウ操作・フォーカス検知・クリップボードが標準機能 |
| 配布形式 | `.ahk` スクリプト（将来的にexe化も可） | AHKインストール済み環境ならそのまま実行可能 |

## アーキテクチャ

```
VoiceDropper.ahk（単一ファイル）
  │
  ├── GUI定義（メモウィンドウ）
  │     ├── Edit コントロール（テキスト入力）
  │     └── Button コントロール（手動コピー）
  │
  ├── フォーカス監視タイマー（SetTimer）
  │     ├── 100ms間隔でアクティブウィンドウを監視
  │     ├── 対象ウィンドウへの切り替えを検知
  │     └── メモ内容をクリップボードにコピー
  │
  └── コピー通知（背景色フラッシュ）
```

## フォーカス検知の実装方針

- `SetTimer` で 100ms 間隔のポーリングを行う
- `WinActive()` で対象ウィンドウがアクティブになったかを判定
- 前回のアクティブウィンドウIDを保持し、「切り替わった瞬間」のみコピーを発火する（連続コピー防止）

## 対象ウィンドウの判定

```autohotkey
; ウィンドウタイトルに以下を含むものを対象とする
targetTitles := ["Chrome Remote Desktop", "Chromeリモート デスクトップ"]
```
